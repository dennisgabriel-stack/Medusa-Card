# Medusa Hookah Lounge - Digitale Speisekarte

Interaktive digitale Speisekarte fuer die Medusa Hookah Lounge. Mobile-first Design, optimiert fuer QR-Code Zugang.

## Setup

```bash
npm install
npm run dev
```

## Logo-Bilder hinzufuegen

Platziere deine Logo-Dateien im Ordner `public/images/`:

- `medusa-logo.png` - Grosses Logo (MEDUSA HOOKAH LOUNGE) fuer den Hero-Bereich (empfohlen: 560x560px)
- `medusa-logo-small.png` - Kleines Logo (nur MEDUSA Kopf) fuer Watermarks auf Karten (empfohlen: 96x96px)

## Vercel Deployment

1. Repository mit deinem Vercel-Konto verbinden
2. Vercel erkennt Next.js automatisch
3. Keine weitere Konfiguration noetig - einfach deployen

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- Framer Motion
- TypeScript
